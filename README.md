# 📊 Dashboard de Recursos Humanos - Power BI

## 📌 Descripción

Este proyecto consiste en un dashboard interactivo de Recursos Humanos desarrollado en Power BI, donde se analizan empleados, sueldos, evaluaciones y distribución geográfica en Argentina.

## 📊 Principales métricas

* Total de empleados: 194
* Edad promedio: 45 años
* Sueldo promedio: 74.72k
* Evaluación promedio: 7.56

## 📍 Visualizaciones

* Mapa de empleados por ubicación
* Distribución por género
* Empleados por grupo etario
* Empleados por departamento
* Tabla detallada de empleados

## 🛠️ Tecnologías utilizadas

* Power BI
* DAX
* Power Query

## 📎 Archivos

* dashboard.pbix: archivo principal
* dataset.xlsx: datos utilizados
